Turn any gallery into a puzzle!

== Description ==

1. Add your [gallery] shortcode to a post as normal.
2. Add a puzzle attribute e.g. [gallery puzzle="1" ] 

This plugin registers the gallery shortcode and then delegates to the standard shortcode (or photoswipe-masonry if installed) if no puzzle is requested.

== Installation ==

This section describes how to install the plugin and get it working.

1. Upload the plugin files to the `/wp-content/plugins/plugin-name` directory, or install the plugin through the WordPress plugins screen directly.
1. Activate the plugin through the 'Plugins' screen in WordPress
1. No Settings->Plugin Name screen is needed to configure the plugin
